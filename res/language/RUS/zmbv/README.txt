Начиная с версии 0.65, DOSBox позволяет вам создавать ролики, записывая экран.

Для записи ролика вы должны нажать CTRL-ALT-F5.
Для остановки/конца записи вы должны нажать CTRL-ALT-F5 снова.

Для проигрывания записанного ролика вам понадобится проигрыватель, который может
работать с кодеком ZMBV. Пользователи MS Windows могут найти этот кодек в пуске,
в секции DOSBox.  
Пользователи Linux и других ОС должны найти проигрыватель, который использует
библиотеку ffmpeg (возможно вам понадобится обновление или запрос к
распространителю за апгрейдом).


FAQ:
В: Во время проигрывания ролика звук лагает.
О: Проверьте ваши настройки экрана, чтобы убедится, что в обновлении экрана
стоит по крайней мере 70 hz. Попробуйте проиграть ролик в программе virtualdub  
(http://virtualdub.sf.net)

В: Почему конечный ролик состоит из нескольких файлов?
О: Каждый раз при изменении разрешения игрой, DOSBox создает новый файл ролика,
потому что ролик может содержать только одно разрешение.

В: Могу ли я поставить число циклов CPU больше, чем мой
компьютер может обработать во время записи?
О: Да. Во время записи игра может работать медленно и заикаться, но конечный
ролик должен проигрываться с планированной скоростью и без заиканий.

В: CTRL-ALT-F5 переключает на консоль под linux.
О: 1. Запустите DOSBox таким образом: dosbox -startmapper
   2. Кликните на Video, кликните на Add
   3. Нажмите подходящую вам кнопку (например, scroll lock или printscreen)
   4. Нажмите exit.
   5. Вы сможете делать ролики нажатием scroll lock или любой выбранной кнопкой.
